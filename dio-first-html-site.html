Tags Simples

(Voltar)

Apesar de estarem aqui na lista de tags simples, essas tags são extremante importantes e usuais, você vai conseguir mensurar o poder que essas tags tem quando souber o que cada uma faz.

H's

As Tags que têm por nome o 'H' são tags de Título, começando pela h1 e terminando na h6, grande diferença entre eles é o tamanho do título, sendo h1 o maior e h6 o menor. Inclusive, no título esse tópico, eu usei um h3!

P

A tag P, serve única e exclusivamente para dizer que temos um parágrafo!

Mark

A Tag Mark, vai ser utilizada quando quisermos destacar um pedaço de texto, igual isso aqui!

I

A Tag I, servir para colocarmos o texto em itálico.

Strong

O Strong é a tag que vai deixar o nosso texto em negrito, bem forte!

Blockquote

O Blockquote, você vai utilizar quando quiser colocar uma citação, um destaque legal no texto, tipo isso.

Tá vendo? Fico maneiro, né?

U

Por último mas não menos importante, a tag U, ela serve para colocarmos um underline.

Listas

(Voltar)

Aqui nós vamos falar sobre as listas, na verdade temos duas, a Lista ordenada (OL) e a lista não ordenada, a (UL).

Vale ressaltar que sempre que você quiser colocar uma lista, é obrigatório o uso dos itens nessas listas, que é representado pelo (IL).

OL

Como dito anteriormente, a OL, é a Lista Ordenada, nesse caso, os itens da lista serão representados por números, porque haverá uma ordem.

UL

Já o UL, que é a Lista não Ordenada, será usada para listas que não tem uma ordem correta, como se fosse um menu ou o Índice, que está lá em cima do nosso site!

LI

LI, são os itens de uma lista. Lembra que eu falei ali em cima que é obrigatório ter os itens para que seja possível adicionar uma OL ou uma UL? Então, o LI é exatamente isso.

Adições interessantes

(Voltar)

Quando falamos de HTML, podemos combinar as nossas Tags com adições que tornam o nosso site mais interessante e chamativo, vou deixar aqui abaixo algumas que são simples mas julgo bem legais de se usar!

A

A Tag 'A', serve pra gente colocar um link, mas não apenas um link para outra página, a gente consegue colocar um link pra dentro da mesma página, como se fosse uma seleção de um conteúdo específico.

Com certeza você já clicou em algum lugar de um site ou artigo que te levou exatamente para onde o assunto estava sendo mencionado, aqui não vai ser diferente, você vai perceber isso no nosso primeiro site!

HR

O HR é bem simples, ele adiciona uma linha em determinado local, como uma espécie de espaçador.

Sub e Sup

Esses dois são bem semlehantes embora diferentes... Um serve para a palavra ficar em baixo e o outro pra ficar em cima.
